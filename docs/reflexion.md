## Selbstreflexion Benjamin

### Positive Punkte

Das ganze Projekt war an sich spannend, ich durfte zum ersten mal einen wirklichen Cloud Service wie AWS benutzen. So habe ich ein wenig erfahrung gesammelt.

Ich hatte mich so gefreut nach dem alles so funktioniert hatte wie ich das erhoffte. Mit diesem Projekt habe ich die Basics von AWS kennengelernt und kann sie später wiederverwenden wenn ich wieder AWS oder einen anderen Cloud Service benutze.

Den Python Skript hatte ich dank ein wenig Python vorwissen  relativ schnell fertig, alle notwendige konfigruationen waren auch keine grosse Sache.

### Negative Punkte

Am anfang des Projekts habe ich die aufgabenstellung nicht ganz verstanden, und bin deswegen zu lange steckengeblieben. Aber daraus habe ich zumindest mal gelernt das ich früher bei unklarheiten nachfragen soll.

Die grösste unklarheit die ich hatte, war wie das ganze mit dem Init Skript in AWS zusammenhängt. Aber mit der hilfe von einer Lehrperson wurde mir das alles sofort klar.

Ab und zu war auch AWS irgendwie nervig zu benutzen, da es manchmal zu lange brauchte zum laden, oder die berechtigungen nicht stimmen.
